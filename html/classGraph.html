Adds a directed edge from source to edge. Returns a pair, the edge descriptor and a boolean. true if the add was successful, false if not. Will return false if the edge already exists from source and dest, the existing edge_descriptor is returned Does not allow edges where source == dest. If source or dest > vertices(), the new size will be max(source, dest) + 1 and new vertices will be created to fill the new size.
